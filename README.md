# Huberisation

Code source du blog [huberisation.eu](https://huberisation.eu)

## Dépendances

* [Hugo](gohugo.io/)

## Lancement

Lors du développement, utiliser la ligne de commande suivante :
``
hugo server -D
``

Pour construire les fichiers finaux de l'application, utiliser la ligne de commande : 
``
hugo -D
``
